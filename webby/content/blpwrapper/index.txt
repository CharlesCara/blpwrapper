org.findata.blpwrapper is a Java package to wrap the Bloomberg API.

blpwrapper is used in the RBloomberg R package. Examples are available for incorporating blpwrapper into JRuby and Jython.

To obtain the source code you can

@bzr branch http://findata.org/code/blpwrapper@
